# Twin Ravens
  
Travel planning with friends with real-time updates on location and ETA.
  
## Dev Setup

```
pip install -r requirements.txt
npm install
./manage.py migrate
./manage.py loaddata data.json
npm start
```

Open another terminal and do:

```
./manage.py runserver
```
